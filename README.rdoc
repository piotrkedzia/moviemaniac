== MOVIE MANIAC

This application is based on Pragmatic Studio Ruby on Rails level I course: https://pragmaticstudio.com/rails 

I've aded few additional features and gems:

* Upgrade to Ruby on Rails 4.2.2

* Authentication with Devise

* Forms rewritten to Simple Forms

* View templates rewritten to HAML

* Bootstrap front-end

* View helpers moved to proper decorators Draper

* Few JQuery effectcs

* Time ago in words computed by Moment.js

Develop server:
http://movmaniac.herokuapp.com
